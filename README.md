# TravelPlanner

Bеб-сервис для планирования поездок и бронирования туров.  
Стек: **Jakarta Servlets + FreeMarker + PostgreSQL + HikariCP**.

## Функционал

### Пользователь
- Регистрация и вход в систему.
- Просмотр каталога туров.
- Бронирование тура.
- Просмотр своих бронирований в профиле.
- Оставление отзывов и оценок к турам.

### Администратор
- Управление турами (создание, редактирование, удаление).
- Просмотр бронирований.
- (Опционально) управление пользователями, модерация отзывов, простая статистика.

## Технологии

- Java 17 (или твоя версия из `pom.xml`)
- Jakarta Servlet API 5
- FreeMarker
- PostgreSQL
- HikariCP
- Maven

## Требования

- JDK 11+
- Maven
- PostgreSQL
